Projects

Discord Dice Rolling Bot

Built using Python, this bot takes various commands from a user to simulate rolling multiple dice. The bot then displays a newly-rendered composite image from a library of custom dice images to display the results. The bot can save the roll values and allow the user to re-roll a specific die and render a new result image. In addition to displaying an image of the results, the bot also adds together the die values and displays a numeric total. This is designed for the new Marvel Multiverse RPG game and will be available for purchase on discord soon!
